# Set Up a Customer Site with Experience Cloud

## Project Overview
This project demonstrates the setup of a customer-facing portal using Salesforce Experience Cloud. 
The site allows customers to securely log in and access relevant data based on assigned permissions.

## Technologies Used
- Salesforce Experience Cloud
- Salesforce Admin Configuration
- Experience Builder
- User & Permission Management

## Features Implemented
- Created an Experience Cloud customer site
- Configured customer users and profiles
- Assigned roles and permission sets
- Customized pages using Experience Builder
- Tested secure login and data visibility
